Présentation

La mise à jour 3.4.2 est une mise à jour de service du logiciel/microprogramme In-Sight Explorer. Dans cette mise à jour, aucune fonction n'a été ajoutée au serveur In-Sight OPC Server et aucun des bogues n'a été corrigé. En ce qui concerne la configuration système requise et les problèmes connus, veuillez consulter les notes de mise à jour du microprogramme In-Sight OPC Server 3.4.0.


Pour obtenir les dernières notes de mise à jour, la documentation et autres, connectez-vous au Centre de support en ligne In-Sight.
