Resumen

La versión 3.4.2 es una versión de servicio del software/firmware de In-Sight Explorer. En esta versión no se han añadido funciones a In-Sight OPC Server ni se han corregido errores. Para obtener información sobre los requisitos del sistema y los problemas conocidos, consulte las notas de la versión 3.4.0 de In-Sight OPC Server.


Para obtener las notas, documentación y demás información más recientes sobre esta versión, inicie una sesión en el Centro de asistencia en línea de In-Sight.
