Übersicht

Version 3.4.2 ist eine „Service-Release“ der In-Sight Explorer-Software/Firmware. Dem In-Sight-Anzeigesteuerelement wurden in dieser Version keine Funktionen hinzugefügt, und es wurden keine Defekte des Anzeigesteuerelements behoben. Informationen über Systemanforderungen und bekannte Probleme finden Sie in den Versionshinweisen zum In-Sight-Anzeigesteuerelement 3.4.0.


Diese Version des In-Sight-Anzeigesteuerelements umfasst keine API-Änderungen gegenüber Version 3.4.0 oder Version 3.4.1. Allerdings sind mit einer anderen Version von In-Sight Display Control entwickelte COM/ActiveX-Anwendungen nicht mit dieser Version kompatibel und funktionieren nicht vorschriftsmäßig. Entweder muss die benutzerdefinierte Anwendung für diese Version neu kompiliert oder die Originalversion des Anzeigesteuerelements neu installiert werden.


Aktualisierte Versionshinweise, Dokumentation und mehr finden Sie im In-Sight Online Support Center.
