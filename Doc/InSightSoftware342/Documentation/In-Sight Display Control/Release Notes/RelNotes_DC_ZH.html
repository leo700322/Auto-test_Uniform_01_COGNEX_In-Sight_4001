In-Sight® Display Control 3.4.2 发行说明

2007 年 7 月 16 日 16 时 58 分

1999-2007年Cognex Corporation 版权所有。

概述

版本 3.4.2 是 In-Sight 浏览器软件/固件的服务版本。此版本未增加 In-Sight Display Control 的功能，也未修复 Display Control 中的任何错误。有关“系统要求”和“已知问题”，请查阅“In-Sight Display Control 3.4.0 发行说明”。


此版本的 In-Sight Display Control 不包含 3.4.0 版本或 3.4.1 版本的 API 更改；不过，使用不同版本 In-Sight Display Control 开发的 COM/ActiveX 应用程序与此版本不兼容，无法正常工作。需要针对此版本重新编译自定义应用程序，或必须重新安装原始版本的 Display Control。


有关更新的发行说明、文档及其它信息，请登录到 In-Sight 联机支持中心。
