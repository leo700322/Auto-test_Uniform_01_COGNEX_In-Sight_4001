概要

リリース 3.4.2 は、In-Sight Explorer ソフトウェアおよびファームウェアのサービスリリースです。このリリースでは新機能の追加や不具合修正は行われていません。システム要件や既知の問題については、In-Sight ディスプレイコントロール 3.4.0 のリリースノートをご参照ください。


In-Sight ディスプレイコントロール 3.4.2 の API は、3.4.0 および 3.4.1 リリースと同一です。しかし、このバージョン 3.4.2 と完全互換性のない、ほかのバージョンの In-Sight ディスプレイコントロールを使って開発された COM/ActiveX アプリケーションは、バージョン 3.4.2 では正常に動作しない可能性があります。カスタムアプリケーションをバージョン 3.4.2 用にコンパイルし直すか、あるいは、適切なバージョンの In-Sight ディスプレイコントロールを再インストールする必要があります。


更新されたリリースノートやマニュアル等は、In-Sight オンラインサポートセンター で入手できます。

リリース履歴

これまでの API の変更点については、In-Sight ディスプレイコントロールのヘルプファイルをご参照ください。
