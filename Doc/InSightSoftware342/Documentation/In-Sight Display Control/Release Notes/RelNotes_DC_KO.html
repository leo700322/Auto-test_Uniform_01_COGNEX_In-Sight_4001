개요

릴리스 3.4.2는 In-Sight 익스플로러 소프트웨어/펌웨어의 서비스 릴리스입니다. 이 릴리스의 In-Sight 디스플레이 제어에는 추가된 기능이 없으며 디스플레이 제어에서 수정된 버그도 없었습니다. 시스템 요구 사항 및 알려진 문제점에 대한 내용은 In-Sight 디스플레이 제어 3.4.0 출시 안내서를 참조하십시오.


In-Sight 디스플레이 제어의 이 릴리스에서는 3.4.0 릴리스 또는 3.4.1 릴리스에서의 API 관련 사항은 변경되지 않았으나, 다른 버전의 In-Sight 디스플레이 제어를 사용해 개발된 COM/ActiveX 응용 프로그램은 이 버전과 호환되지 않으므로 올바르게 작동하지 않습니다. 이 버전에 대해 사용자 정의 응용 프로그램을 다시 컴파일하거나 디스플레이 제어의 원래 버전을 다시 설치해야 합니다.


업데이트된 출시 안내서, 문서 등에 대한 정보는 In-Sight 온라인 지원 센터로 로그온하십시오.



릴리스 내역

이전 릴리스의 특정 API 변경 사항은 In-Sight 디스플레이 제어 참조 도움말 파일을 참조하십시오.
