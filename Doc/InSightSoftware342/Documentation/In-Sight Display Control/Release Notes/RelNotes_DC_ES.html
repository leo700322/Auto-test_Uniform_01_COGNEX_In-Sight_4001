Resumen

La versión 3.4.2 es una versión de servicio del software/firmware de In-Sight Explorer. En esta versión no se han corregido errores del control de pantalla de In-Sight ni se le han añadido funciones. Para obtener información sobre los requisitos del sistema y los problemas conocidos, consulte las notas de la versión 3.4.0 del control de pantalla de In-Sight.


Esta versión del control de pantalla In-Sight no contiene modificaciones en la API con respecto a las versiones 3.4.0 y 3.4.1; sin embargo, las aplicaciones COM/ActiveX desarrolladas con una versión diferente del control de pantalla de In-Sight no son compatibles con esta versión y no funcionarán correctamente. Para que las aplicaciones personalizadas funcionen correctamente, será necesario recompilarlas específicamente para esta versión o volver a instalar la versión original del control de pantalla.


Para obtener las notas, documentación y demás información más recientes sobre esta versión, inicie una sesión en el Centro de asistencia en línea de In-Sight.
