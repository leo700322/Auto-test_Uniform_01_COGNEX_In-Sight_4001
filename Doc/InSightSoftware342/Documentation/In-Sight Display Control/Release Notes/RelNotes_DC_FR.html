Présentation

La mise à jour 3.4.2 est une mise à jour de service du logiciel/microprogramme In-Sight Explorer. Dans cette mise à jour, aucune fonction n'a été ajoutée au contrôle In-Sight Display Control et aucun des bogues du contrôle Display Control n'a été corrigé. En ce qui concerne la configuration système requise et les problèmes connus, veuillez consulter les notes de mise à jour In-Sight Display Control 3.4.0.


Cette mise à jour du contrôle In-Sight Display Control ne contient pas de modifications de l'interface API par rapport à la version 3.4.0 ; cependant, les applications COM/ActiveX développées à l'aide d'une autre version du contrôle In-Sight Display Control ne sont pas compatibles avec cette version et ne fonctionneront pas correctement. Les applications personnalisées doivent de nouveau être compilées en fonction de cette version ou la version originale du contrôle Display Control doit être réinstallée.


Pour obtenir les dernières notes de mise à jour, la documentation et autres, connectez-vous au Centre de support en ligne In-Sight.
